# Hangman

A code to play hangman

## Instructions for Build and Use

Steps to build and/or run the software:

1. Clone repository
2. Open in a code editor that can run c++
3. Debug and run

Instructions for using the software:

1. you will guess letters till you either run out of lives or get it right
